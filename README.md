## 概要

3D AR Viewerは、STL・GLB形式の3Dモデルファイルを読み込み、カメラ映像や静止画像と合成して表示するARツールです。TinkercadなどのCADツールで作成した3Dモデルを、実際の写真や風景の上に重ね合わせることができます。

### 主な特徴

- STL・GLB（GLTF）形式の3Dモデル読み込み対応
- カメラ映像または事前撮影した静止画像を背景として合成
- モデルの移動・回転・拡大縮小（OrbitControls によるインタラクティブ操作）
- 前面カメラ・背面カメラの切り替え対応
- 合成画像をJPEGで画面キャプチャ・ダウンロード
- STLモデルの部品別色分け表示（複数部品を自動識別・カラーリング）
- ポリゴン（ワイヤーフレーム）表示切替
- モデルの統計情報表示（ポリゴン数・頂点数・モデルサイズ等）
- GIGA端末・スマートフォン・iPad・PC対応のレスポンシブデザイン

### 教育利用

本ツールは以下の教育用途を想定しています：

- STEAM教育におけるデジタルファブリケーション学習
- TinkercadなどのCAD作品を現実空間でプレゼンテーション
- 3Dモデルの構造・部品構成の視覚的確認
- 材料加工（木工・プラスチック加工等）の設計前確認

## 起動方法

4つのファイル（`index.html`、`main.js`、`style.css`、`botton.png`）を同じフォルダに置き、`index.html` をブラウザで開いてください。

> **対応ブラウザ：** Chrome / Edge / Firefox / Safari（最新版推奨）  
> **インストール不要：** CDN経由でライブラリを読み込むため、ネットワーク接続が必要です  
> **対応端末：** Windows、Mac、iPad、Chromebook、Android、iPhone

## 操作手順

### 1. モデルを読み込む

スタート画面の画像をクリック後、「モデル読み込み」ボタンをクリックしてSTL・GLBファイルを選択します。  
ファイルのドラッグ＆ドロップにも対応しています。

### 2. モデルを操作する

| 操作 | 動作 |
|---|---|
| マウスドラッグ（左ボタン） | モデルを回転 |
| マウスドラッグ（右ボタン） | モデルを平行移動 |
| マウスホイール | 拡大・縮小 |
| モデルをクリック＆ドラッグ | モデルを画面上で移動 |

### 3. 背景を設定する

| ボタン | 動作 |
|---|---|
| 🖼 静止画背景 | 画像ファイルを読み込んで背景に設定 |
| 📹 カメラ背景 | デバイスのカメラ映像を背景に設定（背面カメラ優先） |
| 🔄 カメラ切替 | 前面・背面カメラを切り替え |
| ✕ 背景リセット | 背景を黒に戻す |

### 4. モデルを確認・分析する

| ボタン | 動作 |
|---|---|
| ⬡ ポリゴン表示 | ワイヤーフレーム表示のON/OFF切り替え |
| 📊 統計情報 | メッシュ数・ポリゴン数・頂点数・モデルサイズを表示 |
| 🎨 STL部品区別 | STLモデルを部品（連結成分）ごとに色分け表示（STLのみ） |

### 5. 保存する

「📷 キャプチャ」ボタンで画面全体（モデル＋背景合成）をJPEG形式でダウンロードします。  
ファイル名は `capture_YYYYMMDD_HHMMSS.jpg` の形式で保存されます。

## STL部品区別機能

STLファイルを読み込んだ際に「🎨 STL部品区別」ボタンが表示されます。

- ボタンを押すと、STLメッシュを**連結成分（部品）ごとに分離**し、12色のパステルカラーで色分けして表示します
- もう一度押すと元の単色表示に戻ります
- 部品数はボタンに `部品区別中 (N個) ／元に戻す` として表示されます
- 色分け表示中もドラッグ移動・回転・拡大縮小・ポリゴン表示切替が通常通り使用できます

## 統計情報の表示項目

| 項目 | 説明 |
|---|---|
| メッシュ数 | モデルを構成するメッシュの数 |
| マテリアル数 | 使用されているマテリアルの数 |
| 三角形数 | ポリゴン（三角形）の総数 |
| 頂点数 | モデルを構成する頂点の総数 |
| ドローコール | 描画命令の回数の目安 |
| モデルサイズ | W・H・D（単位：mm、Tinkercadで作成した場合） |

## 対応ファイル形式

| 形式 | 拡張子 | 特徴 |
|---|---|---|
| STL（バイナリ・ASCII両対応） | `.stl` | 単色表示。部品区別機能で色分け可能 |
| GLTF/GLB | `.glb` `.gltf` | テクスチャ・色情報を含む高品質表示 |

## 使用ライブラリ

外部ライブラリとして [Three.js](https://threejs.org/) を CDN 経由で使用しています。

| ライブラリ | 役割 |
|---|---|
| `three.module.js` | 3Dシーンの構築・描画の基本機能 |

| `OrbitControls.js` | マウス・タッチによる直感的な3D操作 |

## レスポンシブ対応

CSS変数と `clamp()` 関数を使用し、画面サイズに応じてフォントサイズ・ボタンサイズが自動調整されます。

| 対象端末 | ブレークポイント |
|---|---|
| 大画面PC | 1200px 以上 |
| 標準デスクトップ | 769px〜1199px |
| タブレット縦向き | 768px 以下 |
| スマートフォン縦向き | 480px 以下 |
| 横向き・低背画面 | 高さ 480px 以下 |

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| モデルが表示されない | 対応ブラウザ（Chrome推奨）で開いているか確認してください |
| カメラが起動しない | ブラウザのカメラアクセス許可を確認してください |
| キャプチャが保存されない | ブラウザのポップアップ・ダウンロード許可を確認してください |
| GLBが正しく表示されない | ファイルが破損していないか、対応形式か確認してください |
| STL部品区別ボタンが表示されない | STLファイル読み込み時のみ表示されます |
